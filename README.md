automatizador lançador de faltas automatico.

Imagine que voce tem uma planilia de faltas  e precise alimentar um sistema interno. Entao imagine que voce tem um programa que tem a capacidade de coletar um campo desta planilia, logicamente e previamente preenchido com faltas, para lançar em um sistema, entao imagine que voce tem um "lançador" que faz isto automatico pra voce.

se ele ler C ele nao faz nada.

se ele ler 1f ele adicionara uma falta.

se ele ler 1j ele adicionara uma falta justificada.

Cada sistema tem uma logica diferente para operar atravez do teclado, e voce pode alterar a logica do preenchimento, entao torna-se possivel adaptaçao e atomatizaçao do processo.
